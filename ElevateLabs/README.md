# Responsive Landing Page

## Project Overview

This project is a modern responsive landing page developed using HTML5 and CSS3 as part of a Web Development Internship Task. The website is designed to provide a clean and engaging user experience across desktop, tablet, and mobile devices.

## Features

* Responsive navigation bar
* Hero section with image and call-to-action button
* About Us section
* Features section using CSS Grid
* Statistics section
* Call-to-Action (CTA) section
* Footer with social media links
* Mobile-friendly design using Media Queries
* Smooth hover effects and modern UI styling

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
